   Отчет.игсОборотыДляБДР

ОбщиеМодули:
    игсОборотныйДляБДР:
        ТекстЗапроса() - возвращает текста запроса для отчета;
		КонстантныеПараметры() - возвращает предопределенные параметры для запроса.

Отчеты:
	игсОборотыДляБДР (Этот отчет больше не является внешним!):
		Реквизиты:
			ДатаНачала - тип Дата
			ДатаОкончания - тип Дата
			Организация - тип СправочникСсылка.Организация
			СписокСвертки - тип СписокЗначений
			СписокСчетов - тип СписокЗначений
			СписокПроектов - тип СписокЗначений
			ЭтоДоЗагрузкаРезерва - тип Булево
			ТолькоНП - тип Булево
		Формы:
			ФормаОтчета - основная форма отчета
			ФормаНастроек - основная форма настроек

		Макеты:
			ОсновнаяСхемаКомпоновкиДанных - основная СКД
			
		Модули:
			МодульОбъекта:
				ПриКомпоновкеРезультата() - процедура подтягивает необходимые данные из общего модуля для построения запроса и из формы отчета. Потом данные уходят набором в СКД;
				ПолучитьВидеРезерваУбытПроектов() - возвращает указанный Вид Резерва из НастройкиМеханизмов для указанной Организации в аргументе.
			
			МодульФормы "ФормаОтчета":
				Область "ПрограммныйИнтерфейс":
					1. ПолучитьРезультатСтрокой() - возвращает результат ТаблицаРезультата, сформированную на основе параметров/отборов, в формате строки;
					2. СкопироватьОтборКомпоновкиДанных() - переносит отбор от Источника к Приемнику;
					3. ДобавитьЭлементВКоллекцию() - добавляет элемент в указанную коллекцию с указанным типом;
					4. ЗагрузитьНастройки() - позволяет установить в форме, открываемой модально, отборы из текущего мэппинга.
					
				Область "ОбработчикиСобытий":
					1. ПриСозданииНаСервере() - заполняет параметры базовыми значениями, если это возможно;
					2. ПередЗакрытием() - позволяет избегать окна о сохранении варианта;
					3. ПериодПриИзменении() - вызывает ПериодПриИзмененииНаСервере();
					4. ПериодПриИзмененииНаСервере() - заполняет реквизиты отчета.
					
				Область "КомандныйИнтерфейс":
					1. ВыбратьОтбор() - возвращает текущий установленныйй отбор для применения его в Мэппинге.
					
				Область "СлужебныйПрограммныйИнтерфейс":
					1. ПрочитатьНастройкиСтрокой() - функция проверяет полученные настройки на корректность;
					2. ДобавитьВОтборГруппуЭлементовОтбора() - добавляет в отбор группу элементов отбора;
					3. ДобавитьВОтборЭлементОтбора() - добавляет в отбор элемент отбора;
					4. СформироватьТаблицуЗначений() - возвращает результат компоновки (таблица значений в структуре);
					5. ТипЗначенияСтрокой() - возвращает тип значения строкой;
					6. ПолучитьВнешнийНаборДанных() - возвращает набор данных из внешнего модуля;
					7. ПолучитьВидеРезерваУбытПроектов() - возвращает установленный вид резерва под убыток для указанной Организации;
					8. УстановитьОтборСчетЗатрат() - программно устанавливает отбор СчетЗатрат;
					9. УстановитьОтборПроектов() - программно устанавливает отбор Проектов;
					10. УстановитьОтборДоЗагрузкаРезерва() - программно устанавливает отбор ДоЗагрузкиРезерва;
					11. ВыбратьОтборСервер() - возвращает установленный отбор.
					
				Область "Отладка":
					1. ОтладкаПолученияСтроки() - вызывает ОтладкаПолученияСтрокиНаСервере();
					2. ОтладкаПолученияСтрокиНаСервере() - внутренний инструмент отладки, позволяющий получать результат, сформированный при помощи ПолучитьРезультатСтрокой().
