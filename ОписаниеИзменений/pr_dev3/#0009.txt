ВнешниеОбработки:
	игсРаспределениеЗатратПоФлотам - новая обработка, предназначена для создания документов распределения:
		Структура:
			"Дата" - реквизит типа Даты без времени
			"Организация" - реквизит типа "СправочникСсылка.Организации"
			"ОперацияБух" - реквизит типа "ДокументСсылка.ОперацияБух"
			
		Формы:
			"Форма" - основная форма обработки
		
		Модули:
			МодульОбъекта:
				1. СведенияОВнешнейОбработке() - содержит информацию об обработке.
				
			Модуль "Форма":
				1. ОперацияБухПриИзменении() - заполняет Дату и Организацию при заполнении ОперацииБух.
				2. ВыполнитьКоманду() - содержит проверки и вызывает следующую серверную процедуру.
				3. ВыполнитьКомандуНаСервере() - создает движения, основываясь на алгоритме распределения. Вызывает следующие три служебные функции, описанные далее.
				4. ПолучитьПринадлежностьКФлоту() - возвращает принадлежность указанного ТС на ДатуСреза.
				5. ПолучитьТекстОсновногоЗапроса() - возвращает основной текст запроса (поиск ЗатратыВыручка к распределению и БазыРаспределения).
				6. ПолучитьТекстЗапросаСоставыФлотов - возвращает текст запроса со срезом состава Флота на дату.
				7. ПолучитьРеквизит() - возвращает указанный реквизит для указанной ссылки.
				8. ЯвляетсяБезусловнымПриРаспределении() - возвращает значение типа "булево", которое отражает принадлежность указанной организации к определенной настройке из "игсНастройкиМеханизмов".
